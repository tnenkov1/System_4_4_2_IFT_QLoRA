Python script for converting .jsonl files into .docx dataset from Instruction Fine-tuning.

-The .jsonl file must contain instruction, input and output.
-The .jsonl files must be placed into the folder "jsonl_files".

-In the .docx file:
instruction = Heading 1
input = //additional text under heading 1
output = Main text under heading 1

-The .docx files will be saved into a folder in the folder "docx_files".
