Python script for converting .jsonl files into .docx dataset from LM Fine-tuning.

-The .jsonl file must contain prompt and completion (valid Json).
-The .jsonl files must be placed into the folder "jsonl_files".

-In the .docx file:
prompt = Heading 1
completion = Text under heading 1

-The .docx files will be saved into a folder in the folder "docx_files".
